​	将一个链表反转，返回头节点。

## 结构

```java
public class Node {

    public int value;

    public Node next;

    public Node pre;

    public Node(int value){
        this.value=value;
    }

}
```

## 实现

```java
class ReverseLink{
    public  Node reverseLink(Node node){

        Node pre=null;
        Node next=null;
        while (node!=null){
            next=node.next;
            node.next=pre;
            pre=node;
            node=next;
        }
        return pre;
    }
}
```
